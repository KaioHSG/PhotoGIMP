# PhotoGIMP Windows

## How to install?

**1**. Install [**GIMP**](https://www.gimp.org/downloads) (2.10+).

**2**. Run `Install PhotoGIMP Windows.bat`.

**3**. Confirm overwriting the previous configuration.
